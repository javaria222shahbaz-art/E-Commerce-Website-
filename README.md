# Responsive E-Commerce Frontend
A clean, modern, and fully responsive e-commerce storefront template. This project focuses entirely on the user interface and client-side experience of an online retail shop.

## Features
* Home & Catalog: Modern landing page with a featured bestseller layout (`index.html`, `bestseller.html`).
* Product Discovery: Complete item grid and detailed single-product views (`shop.html`, `single.html`).
* Shopping Flow: Fully styled shopping cart and checkout pipeline (`cart.html`, `checkout.html`).
* Support Pages: Custom contact form and a branded 404 error page (`contact.html`, `404.html`).

## Tech Stack
* Frontend: HTML5, CSS3, JavaScript
* Libraries: External UI components and icons (stored in `/lib`)
